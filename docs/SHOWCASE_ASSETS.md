# Showcase Assets Guide

Use this folder structure for your GitHub README visuals:

- `docs/screenshots/login.png`
- `docs/screenshots/admin-dashboard.png`
- `docs/screenshots/teacher-dashboard.png`
- `docs/screenshots/student-dashboard.png`
- `docs/screenshots/grades.png`
- `docs/videos/demo.mp4` (optional local archive)

## Recommended screenshot format

- Width: `1365px` or `1920px`
- Format: `png`
- Keep UI language consistent (FR or EN, not mixed)
- Use realistic data (students, courses, grades)

## Suggested capture order

1. Login page
2. Admin dashboard
3. Teacher dashboard
4. Student dashboard
5. Grades page
6. Error page preview (`/debug/403/` and `/debug/404/`)

## Optional polish

- Add one short GIF (8-15 seconds) showing role switch and dashboards.
- Add one architecture image (`docs/screenshots/architecture.png`) if you do not want Mermaid only.
